# Read Me for Math.c
## Author: Ardit Pranvoku
## Does basic math operations given a character from the following and two operands.
## '+' - Does addition
## '-' - Does subtraction
## '/' - Does Division
## '*' - Does Multiplication
## '%' - Modulo
## Example: 
## int Calculate(5, 6, '*')
## output: 30
## Will printf("ERROR") given invalid inputs.
